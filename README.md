# Long Prompting

## Abstract

テキストから画像を生成する技術（Text-to-Image: T2I）は，教育コンテンツ，広告デザインなど幅広い応用が期待される．その中でも，複数の対象間の複雑な空間的・相互作用的な関係を含む長文入力に対応できれば，長文の指示を細部に至るまで正確に反映した画像生成が可能となる．しかし，最新モデルにおいても，こうした詳細な長文入力に対しては，生成画像のテキスト忠実度が低いという課題が報告されている．そこで本研究では，詳細な長文入力に対して，追加学習を必要とせずにテキスト忠実度の向上を可能にする手法を構築し，生成画像の品質向上を図る．

## Datasets
The prompts generated by LLMs for this research are hosted on Hugging Face:
- **Hugging Face Datasets**: [taiyo2001/long-prompting](https://huggingface.co/datasets/taiyo2001/long-prompting)
